📦 PASTA: bot-distribuicao/
────────────────────────────────────────────────────────────────

Contém tudo que o cliente precisa!


🎯 ARQUIVOS PRINCIPAIS
────────────────────────────────────────────────────────────────

INICIAR (Atalho com ícone)
  → Clique aqui para iniciar o bot
  → Usa seu ícone customizado
  → Recomendado para usuário leigo

Run.bat (Arquivo batch)
  → Alternativa ao INICIAR
  → Limpa porta automaticamente se em uso
  → Funciona também

whatsapp-bot.exe (~54 MB)
  → Programa executável único
  → Não precisa de Node.js
  → Funciona em máquina fraca


📁 PASTAS
────────────────────────────────────────────────────────────────

data/
  └─ db.sqlite → Banco de dados (histórico de clientes)

.wwebjs_auth/
  └─ Autenticação do WhatsApp Web

public/
  └─ Interface web (HTML/CSS/JS)


⚙️ CONFIGURAÇÃO
────────────────────────────────────────────────────────────────

config.json
  → Mensagens do bot
  → Opções de menu
  → Palavras-chave (triggers)
  → Edite aqui para customizar!

LEIA-ME.txt
  → Instruções para o cliente


═════════════════════════════════════════════════════════════════
            COMO USAR (Para o Cliente)
═════════════════════════════════════════════════════════════════

1️⃣  INICIAR
   Duplo clique em: INICIAR (ou Run.bat)

2️⃣  ESCANEAR QR CODE
   → Abre janela preta com QR code
   → WhatsApp → Menu → Dispositivos conectados
   → Conectar dispositivo
   → Escaneia o código
   → Confirma no celular

3️⃣  CONFIGURAR
   → Abre navegador
   → Digite: http://localhost:3000/admin
   → Login: admin / admin
   → Edita mensagens
   → Salva

✅ Pronto! Bot funcionando!


═════════════════════════════════════════════════════════════════
            PARA DISTRIBUIR
═════════════════════════════════════════════════════════════════

1. Copie a pasta: bot-distribuicao
2. Customize config.json (opcional)
3. Comprima em ZIP
4. Envie para cliente
5. Cliente extrai e clica em INICIAR

Cada cliente recebe seu próprio:
  ✓ Executável
  ✓ Banco de dados
  ✓ Configuração


═════════════════════════════════════════════════════════════════
            MELHORIAS IMPLEMENTADAS
═════════════════════════════════════════════════════════════════

✅ Executável único (~54 MB)
   → Sem dependências externas
   → Funciona em máquina fraca
   → Chromium já compilado

✅ Atalho com ícone customizado
   → INICIAR aparece com seu ícone
   → Mais profissional e amigável
   → Usuário leigo entende facilmente

✅ Limpeza automática de porta
   → Run.bat mata processo anterior
   → Se porta 3000 está em uso, libera
   → Sem erro "EADDRINUSE"

✅ Detecção de porta disponível
   → index.js busca próxima porta (3001, 3002...)
   → Se 3000 ocupada, usa 3001
   → Múltiplos bots podem rodar simultaneamente

✅ Documentação clara
   → LEIA-ME.txt resumido e organizado
   → Erros comuns resolvidos
   → Instruções passo-a-passo
